As part of the Fortnight 1 homework, we were given the task of creating a drawing with no specific guidelines, allowing for creativity. I chose to incorporate my own poem into my submission, and began thinking about how I wanted to represent the text visually.

I quickly realized that manually plotting each letter using the Latin alphabet would be too time-consuming, so I looked for an alternative method that would allow me to represent the poem with minimal code. Eventually, I settled on using ASCII representation, where each letter is represented by pairs of zeroes and ones. To do this, I created a function to plot zero and one, and then created functions to plot each of the four possible pairs: 00, 01, 10, and 11.
